# TalentLens-AI

TalentLens-AI is an AI project for resume, job description, and recruiter insight workflows.

## Folder Structure

```text
TalentLens-AI/
├── app.py
├── README.md
├── requirements.txt
├── .gitignore
├── docs/
├── modules/
├── data/
│   ├── resumes/
│   ├── job_descriptions/
│   └── outputs/
├── assets/
├── notebooks/
└── venv/
```

## Installation

```bash
pip install -r requirements.txt
streamlit run app.py
```

## Tech Stack

- Python
- Streamlit
- pandas
- NumPy
- scikit-learn
- sentence-transformers
- spaCy
- PyMuPDF
- python-docx
- Plotly
